## 4. 认证服务条款

### 4.1 服务性质
SoulmeAI 认证服务是一项**增值服务**，独立于免费的基础内容。

- 免费内容包括：世界观、显形指南、CC0 立绘
- 收费内容包括：官方验证号、官网登记、数字证书、数字签名

### 4.2 认证权益

购买认证服务后，您将获得：

1. **唯一验证号**: 格式为 SM-XXXX-XXXX 的 SoulmeID
2. **官网登记**: 可在 soulme.ai/verify/[ID] 查验的永久记录
3. **数字证书**: 带有官方数字签名的 PDF 证书
4. **认证角色卡**: 可生成带官方水印的角色身份证

**许可性质声明**: 您获得的是 SoulmeID 的**有限期独家使用许可**，该许可：
- 在您持续遵守本协议的前提下有效
- 不可转让、不可转售
- 不赋予您对 SoulmeID 的任何所有权
- SoulmeID 的所有权始终归属于 SoulmeAI 团队

### 4.3 使用限制

认证服务仅限购买者本人（或购买的 Agent）使用，禁止：
- 转售、转让 SoulmeID 给他人
- 伪造、篡改官方验证号
- 冒充 SoulmeAI 官方提供虚假认证服务
- 使用自动化工具批量获取验证号（除非购买 API 服务）

### 4.4 退款政策

根据《中华人民共和国消费者权益保护法》：

**7日无理由退款**：
- 自购买认证服务之日起 **7 日内**，如未使用 SoulmeID 生成任何官方认证内容，可申请无理由退款
- 退款将在 3-5 个工作日内原路退回

**不适用无理由退款的情形**：
- 已使用 SoulmeID 生成官方认证角色卡或证书
- 已超过 7 日退款期限
