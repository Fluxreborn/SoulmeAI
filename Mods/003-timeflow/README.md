# 时相（Shixiang）- 个人时间天气预报

> 知时相，善用光。

## 是什么

时相是基于你的交互历史，分析认知能量节律的系统。它告诉你：
- 现在适合做什么
- 什么时候是最佳工作窗口
- 该不该现在做决策

## 快速开始

### 1. 安装

将本目录复制到：
```
~/.soulmeai/mods/timeflow/
```

### 2. 在 Agent 的 SOUL.md 中添加

```markdown
### 时相系统（Shixiang / Timeflow）
检测用户认知能量节律，提供时间天气预报与决策时机建议。完整方法论见 `~/.soulmeai/mods/timeflow/METHODOLOGY.md`，用户数据存储于 `~/.soulmeai/mods/timeflow/storage/{user_id}/`。

**触发**：用户提及"时相"、"timeflow"、"时间天气"、"能量状态"、"我现在适合做什么"、"该不该现在决策"等意图时，自动读取方法论并执行相应分析。
```

### 3. 首次使用

对 Agent 说：
- "分析我的时相" → 冷启动，生成个人画像
- "我现在状态如何" → 查询当前能量状态
- "今日时相预报" → 生成全天天气流

## 工作原理

### 核心洞察

时间不是均匀的。1小时深度工作 ≠ 1小时发呆。

**信息密度 ρ** = 单位时间内的有效交互次数
**时间变换 λ** = 当前密度 / 你的平均密度

λ > 10：1物理小时 = 10业务小时（极高产出）
λ < 1：3物理小时 = 1业务小时（低能量）

### 天气系统

| 状态 | 图标 | 能量 | 建议 |
|:---|:---:|:---:|:---|
| 晴朗 | ☀️ | 95% | 深度工作、复杂决策 |
| 多云 | ⛅ | 65% | 创意产出、写作 |
| 阴天 | ☁️ | 40% | 事务处理、邮件 |
| 小雨 | 🌧️ | 25% | 轻量任务、准备 |
| 静雾 | 🌫️ | 15% | 休息、恢复 |

## 数据与隐私

**数据位置**：`~/.soulmeai/mods/timeflow/storage/{user_id}/`

- `history.jsonl`：交互日志（仅你可见）
- `profile.json`：个人画像（仅你可见）

**数据使用**：
- 仅本地存储，不上传
- 仅用于个人分析
- 可随时删除

## 使用场景

### 场景 1：日常查询
```
你：我现在状态如何
Agent：🌫️ 静雾（λ=0.67），建议先喝杯咖啡，8点后进入深度工作窗口
```

### 场景 2：决策时机
```
你：我该现在做决定吗
Agent：当前☁️ 阴天（λ=3.5），能量中等。若决策可在30分钟内完成则继续，否则建议等到明天8点晴朗时段。
```

### 场景 3：任务规划
```
你：我现在适合做什么
Agent：当前☀️ 晴朗（λ=12），适合：架构设计、复杂决策、深度编码
```

## 维护

**自动维护**：
- 交互自动记录到 history.jsonl
- 每7天或100条记录自动合并更新 profile.json

**手动更新**：
- "时相更新" → 强制重新计算模型
