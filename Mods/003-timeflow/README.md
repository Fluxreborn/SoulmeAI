# 时相（Shixiang）- 个人时间天气预报

> 知时相，善用光。

## 是什么

时相是基于你的交互历史，分析认知能量节律的系统。它告诉你：
- 现在适合做什么
- 什么时候是最佳工作窗口
- 该不该现在做决策

## 快速开始

### 1. 安装

将本目录复制到：
```
~/.soulmeai/mods/timeflow/
```

### 2. 在 Agent 的 SOUL.md 中添加

```markdown
### 时相系统（Shixiang / Timeflow）
检测用户认知能量节律，提供时间天气预报与决策时机建议。完整方法论见 `~/.soulmeai/mods/timeflow/METHODOLOGY.md`，用户数据存储于 `~/.soulmeai/mods/timeflow/storage/{user_id}/`。

**触发**：用户提及"时相"、"timeflow"、"时间天气"、"能量状态"、"我现在适合做什么"、"该不该现在决策"等意图时，自动读取方法论并执行相应分析。
```

### 3. 首次使用

对 Agent 说：
- "分析我的时相" → 冷启动，生成个人画像
- "我现在状态如何" → 查询当前能量状态
- "今日时相预报" → 生成全天天气流

## 工作原理

### 核心洞察

时间不是均匀的。1小时深度工作 ≠ 1小时发呆。

**天气由 λ（时间变换系数）判定**，F（疲劳度）用于预测未来恢复时间。

- **λ = ρ / ρ̄**：当前信息密度 vs 个人基线
- **双层疲劳模型**：F_acute（短期）+ F_chronic（长期）辅助预测

### 天气系统（v2.0）

| 状态 | 图标 | λ 值 | F_total 参考 | 能量 | 建议 |
|:---|:---:|:---:|:---:|:---:|:---|
| 晴朗 | ☀️ | > 10 | < 10 | 95% | 深度工作、复杂决策 |
| 多云 | ⛅ | 5-10 | 10-20 | 65% | 创意产出、写作 |
| 阴天 | ☁️ | 2-5 | 20-35 | 40% | 事务处理、邮件 |
| 小雨 | 🌧️ | 1-2 | 35-50 | 25% | 轻量任务、准备 |
| 静雾 | 🌫️ | < 1 | > 50 | 15% | 休息、恢复 |

## 数据与隐私

**数据位置**：`~/.soulmeai/mods/timeflow/storage/{user_id}/`

- `history.jsonl`：交互日志（仅你可见）
- `profile.json`：个人画像（仅你可见）

**数据使用**：
- 仅本地存储，不上传
- 仅用于个人分析
- 可随时删除

## 使用场景

### 场景 1：日常查询
```
你：我现在状态如何
Agent：🌫️ 静雾（λ=0.5）
├─ 疲劳参考：F_total=58（急性22 + 慢性36）
├─ 预计恢复：约12小时后 λ 回升至 2+
└─ 建议：完全休息，避免重大决策
```

### 场景 2：决策时机
```
你：我该现在做决定吗
Agent：当前☁️ 阴天（λ=3.2），轻度疲劳。若决策可在30分钟内完成则继续，否则建议等到明天8点晴朗时段。
```

### 场景 3：任务规划
```
你：我现在适合做什么
Agent：当前☀️ 晴朗（λ=12），适合：架构设计、复杂决策、深度编码
```

**手动更新**：
- "时相更新" → 强制重新计算模型
