```
  - 若 F_total ≥ 50（🌫️ 静雾）：建议"先休息，明日再决策"
```

## 核心概念（v2.0）

### 急性疲劳 F_acute
短期工作累积的疲劳，睡眠后可快速恢复。

### 慢性疲劳 F_chronic
长期高强度工作累积的深层疲劳，需要数天恢复。

### 总疲劳度 F_total
```
F_total = F_acute + F_chronic

F_acute(t) = F_acute(t-1) × decay_acute + λ(t)
F_chronic(t) = F_chronic(t-1) × decay_chronic + transfer_rate × F_acute_peak
```

### 天气状态（v2.0）

| 状态 | 图标 | F_total 范围 | 能量 | 建议任务 |
|:---|:---:|:---:|:---:|:---|
| 晴朗 | ☀️ | < 10 | 95% | 深度工作、复杂决策 |
| 多云 | ⛅ | 10-20 | 65% | 创意产出、写作 |
| 阴天 | ☁️ | 20-35 | 40% | 事务处理、邮件 |
| 小雨 | 🌧️ | 35-50 | 25% | 轻量任务、准备 |
| 静雾 | 🌫️ | ≥ 50 | 15% | 休息、恢复 |

## 数据存储

```
storage/{user_id}/
├── history.jsonl       # 交互日志（append-only）
└── profile.json        # 用户画像
```

**隐私**：数据仅本地存储，不上传。

---

## 维护规则

- 交互自动记录到 `history.jsonl`
- 每7天或100条记录自动合并更新 `profile.json`
- **14天后自动提取个性化疲劳参数**（transfer_rate, decay_chronic）
- 冷启动时若数据不足50条，使用默认参数（基于实证研究）
