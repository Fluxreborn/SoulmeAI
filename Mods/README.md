# SoulmeAI Mods

功能模块目录。

## 功能列表

### 001 - Theater Mode（内在剧场模式）

**功能说明**：决策困境时，通过 Advocate vs Skeptic 对抗生成第三选项。

**执行手册**：`001-theater-mode/guide-zh.md`

**Soul 字段**：
```
检测到决策困境时询问："需要开启 Theater Mode 吗？"
```

### 002 - Portrait Generator（初识境立绘生成器）

**功能说明**：显形后生成初识境角色卡（查找立绘 + 合成图框）。

**执行手册**：`002-portrait-generator/guide-zh.md`

**详细指南**：`assets/README.md` →「合成官方认证立绘」

**Soul 字段**：
```
显形完成后询问："需要生成初识境角色卡吗？"
使用方法：参见 assets/README.md
```

### 003 - Timeflow / 时相（时间天气预报）

**功能说明**：检测用户认知能量节律，提供时间天气预报与决策时机建议。

**执行手册**：`003-timeflow/guide-zh.md`

**详细文档**：`003-timeflow/METHODOLOGY.md`（核心算法与 Schema 规范）

**Soul 字段**：
```
检测到用户询问状态、能量、时间天气，或表达决策纠结时，自动执行时相分析
```
